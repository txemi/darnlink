# Code of Conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)
(version 2.1).

In short: be respectful and constructive. Harassment, discrimination, and personal attacks are not
tolerated. We welcome contributions and questions from everyone.

Report unacceptable behaviour to the maintainer at **txemitron@gmail.com**. Reports are handled
confidentially.
